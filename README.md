A simple Python Script to check YOUR OWN Tokens from Discord with a full capture

---

## 🚀 Features

- **⚡ Multi-Threading:** High-speed verification using concurrent requests (threads).
- **🛡️ Full Capture:** Gives you all the Information Number, email, nitro etc.
- **📊 Clean CLI:** Modern and clean command-line interface (CLI) featuring real-time statistics (Hits, Bad, Free, Errors).
- **📁 Automated Saving:** Valid accounts (Hits) are automatically sorted and saved into separate text files inside the `results/` directory.
- **💻 HTML Gen:** Generates a HTML site for you to view all the Infos .

---

## ❔ How To use

**Simply save you Tokens in a TXT and then start the start.bat file. Once the script is loaded choose the TXT file you saved you
tokens in**
